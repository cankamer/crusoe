This is a intermediate **React Native** project developed to demonstrate advanced mobile development capabilities. Unlike standard To-Do applications that focus solely on future planning, **[Crusoe]** introduces a unique workflow designed to manage, prioritize, and clear **past responsibilities** that often get overlooked in our busy daily lives.

While initially started as a learning initiative, this project implements custom features and performance optimizations not typically found in standard boilerplates.

## ‚ú® Key Features

- **Advanced Animations:** Smooth and complex UI transitions built with **React Native Reanimated**.
